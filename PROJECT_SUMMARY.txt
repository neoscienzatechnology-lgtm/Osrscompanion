================================================================================
API INTEGRATION
================================================================================

OSRS Wiki Prices API:
- Base URL: https://prices.runescape.wiki/api/v1/osrs
- Endpoints:
  • /latest - Current prices
  • /5m?id={itemId} - 5-minute data
  • /mapping - Item mapping
- User-Agent: Required
- Rate limiting: Respected
